# Zyra

Test Your APIs with Zyra

## Rules

- `is` - Test Custom Rule
- `==` - Equal To
- `!=` - Not Equal To
- `>` - Greater Than
- `<` - Less Than
- `>=` - Greater Than or Equal To
- `<=` - Less Than or Equal To
- `in` - Contains
- `not in` - Does Not Contain
- `length >=` - Length Greater Than or Equal To
- `length <=` - Length Less Than or Equal To
- `length ==` - Length Equal To
- `matches` - Regex Match
- `not matches` - Regex Does Not Match
- `has` - Key Exists
- `not has` - Key Does Not Exist

## Built-in Custom Rules

- `is success` - Status Code is 2xx
- `is clientError` - Status Code is 4xx
- `is serverError` - Status Code is 5xx
- `is error` - Status Code is 4xx or 5xx
- `is redirect` - Status Code is 3xx
- `string` - Value is of type String
- `number` - Value is of type Number
- `boolean` - Value is of type Boolean
- `array` - Value is of type Array
- `object` - Value is of type Object
- `null` - Value is Null
- `empty` - Value is Empty
